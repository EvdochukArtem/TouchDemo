TouchScreen Project

Модуль для KernelIP симулирующий работу МФИ25Ш. В модуле реализован ввод данных с 
помощью сенсорных нажатий и/или клавиатуры и мыши. Состав модуля:

-пакет util:
	содержит утилитарные классы обеспечивающие различные функции по управлению
	объектами, распознованию жестов, сетевой обмен и др. Все классы пакета должны
	иметь приватный конструктор и создаваться только классом Util. Класс Util
	реализует шаблон singlton и обеспечивает доступ к остальным утилитарным классам.
	Для сокращения обращения к утилитарному объекту через Util рекомендуется
	использовать #define в StdAfx.h

-пакет object:
	содержит классы представляющие объекты с которыми будет взаимодействовать пользователь
	в процессе работы с приложением. В пакете есть базовые классы (Kadr, Frame, MechanicMenu)
	которые образуют базовый набор элементов в окне приложения. Объекты базовых классов
	должны содержать в себе остальные объекты и осуществлять взаимодействие с ними. Управление
	базовыми объектами и взаимодействие между ними должно осуществляется в EkranHandler.

-TouchScreen:
	точка входа в программы. Содержит глобальные переменные.

-TouchScreenDLL:
	создает окно приложения и получает все поступающие сообщения от системы для приложения.
